The program takes as input a valid Roman numeral from I to MMMMM and convert it to integer equivalent.

. Integer conversion must be based on the following roman numerals and the corresponding values. Integer equivalent values for each literal is provided along with it.
  . I    - 1
  . V    - 5
  . X    - 10
  . L    - 50
  . C    - 100
  . D    - 500
  . M    - 1000
  
. User should provide a valid input in the range I - MMMMM. 
. Only valid roman numerals in the above list are allowed as acceptable inputs
  
. Java version used 16.0.1
. Java Runtime - OpenJDK Runtime Environment
